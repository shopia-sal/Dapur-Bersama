## 1. Alur Utama (Main Flow)

### a. User
- User masuk kedalam Sistem.
- user mencari resep berdasarkan kata kunci, filter, kategori, dan bahan.
- User melihat detail resep, termasuk bahan dan langkah-langkahnya.
- User menulis dan menyimpan resep baru.
- user memberikan ulasan terhadap resep yang sudah dilihat.
- user mengelola profil, seperti mengubah informasi akun.

### b. Admin
- Admin menambahkan resep baru ke dalam sistem.
- Admin mengubah informasi resep yang sudah ada.
- Admin menghapus resep dari sistem.

### c. Sistem
- Sistem menyediakan informasi detail untuk setiap resep.
- Sistem memungkinkan pengguna untuk mencari resep berdasarkan kriteria tertentu.
- Sistem mengkategorikan resep untuk memudahkan penelusuran.
- Sistem memungkinkan pengguna untuk mencari resep berdasarkan bahan tertentu.
